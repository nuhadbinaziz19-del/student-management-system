# 🎓 Student Management System

A desktop-based Student Management System developed using **JavaFX** and **Java File Handling**.  
This application provides a clean and user-friendly interface to manage student records efficiently.

---

# ✨ Features

✅ Admin Login System  
✅ Add Student  
✅ Search Student  
✅ Update Student Information  
✅ Delete Student  
✅ View All Students  
✅ File Handling Data Storage  
✅ Simple & Clean JavaFX GUI
