# 🎓 Student Management System

A desktop-based Student Management System developed using **JavaFX** and **Java File Handling**.  
This application provides a clean and user-friendly interface to manage student records efficiently.

---

# ✨ Features

✅ Admin Login System  
✅ Add Student  
✅ Search Student  
✅ Update Student Information  
✅ Delete Student  
✅ View All Students  
✅ File Handling Data Storage  
✅ Simple & Clean JavaFX GUI

# 🚀 How to Run

## Clone Repository

```bash
git clone https://github.com/your-username/student-management-system-javafx.git
```

## Open Project

Open the project in:

- NetBeans
- IntelliJ IDEA
- Eclipse

## Run Application

Run:

```bash
main.Main
```
